# nTV Platform Scaffolds

HTML5 wrapper apps for TV operating systems that are not covered by the main
Flutter build (Android TV, Apple TV, Roku). Both platforms wrap the nTV web
frontend and are packaged with native TV SDK tooling.

| Platform | Dir | Package | Store | Status |
|---|---|---|---|---|
| Samsung Tizen | `tizen/` | `.wgt` | Samsung Galaxy Store | 🔒 USER-ACTION: account registration |
| LG webOS | `webos/` | `.ipk` | LG Content Store | 🔒 USER-ACTION: account registration |

See each subdirectory's `README.md` for SDK installation, local dev, and
store submission instructions.
